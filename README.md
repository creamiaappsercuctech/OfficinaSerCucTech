# OfficinaPlus PWA

PWA per officine: Accettazione cliente + percorso Causa → Verifica → Riparazione + Audio + Report + Offline.

## Come pubblicarla su GitHub Pages
1. Crea un repo su GitHub (es. `officinaplus-pwa`)
2. Carica tutti i file con la struttura indicata
3. Vai in **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** e folder: **/(root)**
6. Salva. Dopo poco avrai un link pubblico.

## Installazione su Android
- Apri il link GitHub Pages nel browser (Chrome)
- Menu ⋮ → “Aggiungi a schermata Home” oppure usa “Installa”

## Dati procedure
- Default: `data/procedures.json`
- Override sul telefono: pagina `admin.html` (salva in localStorage)
